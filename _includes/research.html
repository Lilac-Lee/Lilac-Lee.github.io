Deterministic PointNetLK for Generalized Registration

    • {% comment %}
  • code
    • {% endcomment %}
We propose a deterministic PointNetLK for generalized registration that uses analytical gradients, which avoid the inherently ill-conditioned finite differentiation and highly sensitive step-size choices in the canonical PointNetLK. Our deterministic PointNetLK provides accurate, reliable registration with high fidelity. Extended experiments on noisy, sparse, and partial point clouds depict the utility of our approach for many real-world scenarios. Further, the decomposition of the Jacobian matrix affords the reuse of feature embeddings for alternate warp functions.

PCRNet: Point Cloud Registration Network using PointNet Encoding
  • arXiv
  • code
  • video
We provide a novel network to align point clouds that utilizes PointNet encoding. Our PCRNet uses data specific information to make accurate registration estimation, which is robust to noise and large initial misalignment between source point cloud and template point cloud. With single pass, PCRNet performs computationally faster than existing methods, while iterative version of PCRNet can refine the registration result to make highly accurate prediction. Our PCRNet can be applied further to 3D tracking, model replacement.

MRS-VPR: A Multi-resolution Sampling-based Global Visual Place Recognition Method
  • ICRA 2019
  • video
    • {% comment %}
  • code
    • {% endcomment %}
In order to perform sequence matching between short-term testing frame and long-term reference frame in visual place recognition, we develop a multi-resolution sampling-based global search method with coarse-to-fine searching and particle filter, which balances the matching accuracy and searching efficiency. By iteratively updating both potential trajectories and frame sequences through low resolution to high resolution, we can find the best match under the highest resolution level.

Mask3D: 3D Object Detection Using Masked VoxelNet
  • proposal
  • paper
  • code
We present a fast, robust, novel model for 3D object detection -- Mask3D, which can also provide additional information of instance reconstruction. Our Mask3D model extends the end-to-end VoxelNet by inserting an additional reconstruction module that takes advantage of deep learning-based point-wise feature learning approach. Our method can improve the prediction accuracy by forcing the deep neural network to generate denser point clouds through utilizing a point cloud reconstruction module.
